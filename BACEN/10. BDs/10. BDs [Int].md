## 1. SQL

- D**D**L (Data **Definition** Language):

  - CREATE
  - ALTER
  - DROP

- D**M**L (Data **Manipulation** Language):

  - INSERT
  - UPDATE
  - DELETE
  - SELECT

- D**Q**L (Data **Query** Language):

  - SELECT

- D**C**L (Data **Control** Language):

  - GRANT
  - REVOKE

- D**T**L (Data **Transation** Language):

  - ROLLBACK
  - COMMIT
