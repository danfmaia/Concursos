### Arquivos .class

- Não contêm código nativo p/ um processador específico.

- Contêm **bytecodes**.

- É uma linguagem da Máquina Virtual Java (JVM).

- São interpretados e executados pelo comando `java` p/ criar uma instância da JVM.
